🔗 Backend-tilkobling

BACKEND URL (Railway)

🚀 Slik setter du opp – steg for steg

  1. Lag gratis konto på railway.app – klikk "New Project" → "Deploy from GitHub"
  2. Last opp backend/-mappen til GitHub, lenk til Railway
  3. Sett Environment Variables på Railway (se under)
  4. Kopier Railway-URL til feltet over og klikk Lagre
  5. Åpne denne HTML-filen i nettleseren og klikk 🚀 Start Agent

🔑 API-nøkler (sett i Railway Environment Variables)

ANTHROPIC_KEY – Claude AI (NØDVENDIG)
Gå til console.anthropic.com → API Keys → Create Key
ANTHROPIC_KEY=sk-ant-api03-...
ETSY_API_KEY + ETSY_ACCESS_TOKEN + ETSY_SHOP_ID
Gå til etsy.com/developers → Create App → OAuth → hent Access Token
Shop ID finner du i URL-en til butikken din: etsy.com/shop/SHOP_ID
ETSY_API_KEY=xxxxxxxxxx
ETSY_ACCESS_TOKEN=xxxxxxxxxx
ETSY_SHOP_ID=12345678
PINTEREST_TOKEN + PINTEREST_BOARD (valgfritt men anbefalt)
Gå til developers.pinterest.com → Create App → Bruk OAuth2 for token
Board ID finner du i URL-en til boardet ditt
PINTEREST_TOKEN=xxxxxxxxxx
PINTEREST_BOARD=1234567890

📱 Instagram & TikTok – semiautomatisk

Instagram og TikTok krever manuelle handlinger (API-restriksjoner). Slik fungerer det:
  1. Agenten genererer perfekte captions og hooks automatisk
  2. Gå til fanen "📋 Godkjenning" – du ser alle klare poster
  3. Klikk ✅ Godkjenn – caption kopieres til utklippstavlen
  4. Lim inn teksten i Instagram/TikTok og post!
⚡ Tips: sjekk godkjenning-fanen én gang om dagen
